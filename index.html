🚀 Quick Start

  1. Upload JSON files using the file inputs or paste JSON directly into the text areas
  2. Click Compare to see the side-by-side diff with Git-style colors
  3. Use the Swap button to quickly switch left and right positions
  4. Toggle the ☀️/🌙 theme button for light/dark mode preference
  5. Scroll through results to see all differences highlighted

🎨 Color Legend

Added - Content only in right JSON
Removed - Content only in left JSON
Changed - Different content in both files
Type Change - Same property, different data type
Unchanged - Identical content

✨ Features

  • 🔒 Privacy-First - Everything runs in your browser, no uploads
  • ⚡ Real-time - Instant comparison as you paste or upload
  • 📱 Responsive - Works on desktop, tablet, and mobile
  • 🎨 Git-style - Professional diff visualization
  • 🌙 Dark/Light Theme - Toggle themes with persistent preference
  • 📁 File Support - Upload .json files or paste directly

⌨️ Command Line

For developers who prefer CLI:

git clone https://github.com/M-Hammad-Faisal/json-compare.git
cd json-compare
npm start file1.json file2.json
